# Glossary (plain language)

- **MicroCT (micro-CT)**: Micro–computed tomography; high-resolution CT imaging typically used for small samples (often ex vivo).
- **HRpQCT**: High-resolution peripheral quantitative computed tomography; high-resolution CT imaging of peripheral sites (e.g., distal radius/tibia).
- **Network visualization**: An interactive graph showing relationships between items (nodes) connected by links (edges).
- **Co-author network**: Nodes are authors; links indicate co-authorship in the dataset.
- **Institution / country network**: Nodes are institutions/countries; links indicate co-occurrence in the dataset.
- **Co-word network**: Nodes are terms/keywords; links indicate co-occurrence across records.
- **Bibliographic coupling**: Links indicate that two papers cite some of the same references.
- **Historical citation network**: Links indicate citation relationships across time.
