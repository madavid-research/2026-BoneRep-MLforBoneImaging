# Networks

This folder contains interactive HTML network visualizations (e.g., co-authors, institutions, co-words).

## Viewing

- Open any `*.html` file in a browser.
- Start with `networks/index.html` for a clickable list of all network pages.
- Some files reference external CDN assets (e.g., `cdnjs` / `jsdelivr`), so viewing may require an internet connection.
- For a short walkthrough, see `docs/VIEW_NETWORKS.md`.

## Archival note

For long-term reproducibility, consider regenerating/exporting “fully self-contained” HTML (bundled JS/CSS) or committing the required local `lib/` assets alongside the HTML outputs.
